Materi

Karena jaring-jaring kubus merupakan 6 buah persegi yang sama dan kongruen maka luas permukaan kubus = luas jaring-jaring kubus = 6 × (s × s) = 6 × s2

Kesimpulan: Luas permukaan kubus = 6(s pangkat 2)

Karena pada balok terdapat 3 buah pasang sisi yang berahadapan sama besar, maka luas bidang sisi balok adalah: a. Sepasang sisi depan dan sisi belakang 2 x p x l b. Sepasang sisi samping kiri dan kanan 2 x l x t c. Sepasang sisi atas dan alas 2 x p x t

Kesimpulan: Luas permukaan balok = 2pl + 2lt + 2pt = 2 (pl +lt + pt)

Luas permukaan limas = luas O.AB + luas O.BC + luas O.CD + luas O.AD + L.ABCD = (luas O.AB + luas O.BC + luas O.CD + luas O.AD) + L.ABCD = jumlah luas sisi tegak + luas alas

Kesimpulan: Luas permukaan limas = jumlah luas sisi tegak + luas alas

Luas permukaan prisma = ( luas ABC + luas EFD) + (luas IADL + luas ACFD + luas CJKF) = ( 2 x luas ABC ) + { ( IA x t ) + ( AC x t ) + ( CJ x t ) } = ( 2 x luas alas ) + { t ( IA + AC + CJ ) } = ( 2 x luas alas ) + ( t x keliling alas )

Kesimpulan: Luas permukaan prisma = ( 2 x luas alas ) + ( keliling alas x tinggi prisma)

Kubus

Luas Alas = sisi x sisi = s x s = s pangkat 2 Volume Kubus = Luas Alas x tinggi = (s pangkat 2) x s

Kesimpulan: Volume Kubus dengan panjang sisi s satuan adalah (s pangkat 3)

Balok

Luas alas = panjang x lebar = p x l Volume balok = Luas alas x tinggi

Kesimpulan: Volume balok = p x l x t

Limas

Volume limas dapat ditentukan dengan membelah sebuah kubus bersisi r menjadi enam buah limas yang kongruen, dimana: Tinggi limas = setengah rusuk kubus t limas =setengah r atau r = 2t limas Maka didapat: Volume kubus = 6 volume limas Volume limas = (1/6) volume kubus = (1/6) x r^3 = (1/6) x (r^2).r Karena r = 2 t limas, maka: Volume limas = (1/6) x (r^2).2t = (1/3) x (r^2).t

Kesimpulan: Volume limas = (1/3) x luas alas x tinggi

Prisma

Volum prisma dapat ditentukan dengan membelah sebuah balok menjadi dua bagian sama besar melalui salah satu diagonal bidang sehingga membentuk dua prisma yang kongruen. 2 Volume prisma = volume balok = p x l x t Volume prisma = 1/2 x p x l x t = ((1/2)×luas alas balok)×t = luas alas prisma x t

Kesimpulan: Volume Prisma = luas alas x tinggi
